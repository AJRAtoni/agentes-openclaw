La colección más completa de agentes IA para OpenClaw traducida y adaptada al español. Cada agente incluye un `SOUL.md` listo para copiar y pegar directamente en tu instancia de OpenClaw.

## ¿Qué es un agente OpenClaw?

Un agente de OpenClaw es un asistente IA especializado definido por su archivo `SOUL.md` — el equivalente a un system prompt estructurado que define identidad, responsabilidades y comportamiento. Con OpenClaw puedes tener múltiples agentes especializados trabajando de forma autónoma o en equipo.

## Cómo usar estos agentes

```bash
# 1. Navega hasta el agente que necesitas
cd agentes/marketing/echo/

# 2. Copia el contenido de SOUL.md
cat SOUL.md

# 3. Pégalo como system prompt en tu instancia de OpenClaw
```

O usa la **[interfaz web interactiva](https://saravrai.com/agentes/)** para buscar, filtrar y copiar en un clic.

## Contribuir

¿Tienes un agente OpenClaw que quieras añadir a la colección?

1. Haz fork del repositorio
2. Crea una carpeta `agentes/{categoria}/{nombre-agente}/`
3. Añade tu `SOUL.md` siguiendo la estructura de los agentes existentes
4. Abre un Pull Request con una descripción del agente

**Criterios de aceptación:**
- Template en español con adaptación cultural (no traducción automática)
- Estructura SOUL.md completa (Identidad, Responsabilidades, Directrices)
- Categoría apropiada o propuesta de nueva categoría
